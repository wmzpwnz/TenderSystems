# Инструкция по получению токена для работы с сервисами ЕИС

## ⚠️ Важно: Изменения с 01.01.2025

С 1 января 2025 года ГИС ЕИС ЗАКУПКИ переходит с FTP-сервера на сервисы отдачи информации и документов в машиночитаемом виде.

## Доступные сервисы

1. **getDocsIP** — для физических лиц (ФЛ)
   - URL: https://int.zakupki.gov.ru/eis-integration/services/getDocsIP

2. **getDocsLE** — для юридических лиц (ЮЛ)
   - URL: https://int44-ttls-cert.zakupki.gov.ru/eis-integration/services/getDocsLE

## Как получить токен

### Шаг 1: Регистрация в личном кабинете

1. Перейдите на страницу открытых данных ЕИС:
   https://zakupki.gov.ru/epz/opendata/search/results.html

2. Найдите кнопку **"Получение открытых данных"** над заголовком **"Результаты поиска"**

3. Нажмите на кнопку и пройдите авторизацию через **ЕСИА** (единая система идентификации и аутентификации)

### Шаг 2: Регистрация в личном кабинете получателя открытых данных

1. После авторизации вы попадете в личный кабинет получателя открытых данных

2. Пройдите регистрацию (если еще не зарегистрированы)

3. После регистрации вам будет предоставлен токен для работы с сервисами

### Шаг 3: Настройка переменных окружения

Скопируйте полученный токен в файл `.env`:

```bash
# Для физических лиц (ФЛ)
EIS_SOAP_TOKEN=ваш_токен_для_физических_лиц

# Для юридических лиц (ЮЛ) - опционально
EIS_SOAP_TOKEN_LE=ваш_токен_для_юридических_лиц

# Тип пользователя: IP (физическое лицо) или LE (юридическое лицо)
EIS_SOAP_USER_TYPE=IP
```
