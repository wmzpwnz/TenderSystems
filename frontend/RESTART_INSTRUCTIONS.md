## Шаги для применения изменений:

### 1. Очистка кеша Vite (уже выполнено)
```bash
npm run clean:cache
# или вручную:
rm -rf node_modules/.vite
```

### 2. Перезапуск dev сервера
Если сервер уже запущен, остановите его (Ctrl+C) и запустите заново:
```bash
npm run dev
```

### 3. Очистка кеша браузера
В браузере:
- **Chrome/Edge**: Ctrl+Shift+Delete (или Cmd+Shift+Delete на Mac)
- Выберите "Кешированные изображения и файлы"
- Или используйте Hard Refresh: Ctrl+Shift+R (Cmd+Shift+R на Mac)

### 4. Проверка работы
1. Откройте консоль разработчика (F12)
2. Измените фильтры поиска
3. Проверьте логи в консоли - должны видеть:
   - `[Dashboard] Search filters changed:`
   - `[Dashboard] Cache key:`
   - `[Dashboard] Calling liveSearch with params:`
4. Список тендеров должен обновляться при изменении фильтров

## Если проблема сохраняется:

### Полная пересборка:
```bash
# Остановите dev сервер
# Очистите все кеши
npm run clean

# Переустановите зависимости (если нужно)
rm -rf node_modules package-lock.json
npm install

# Запустите заново
npm run dev
```

## Новые скрипты в package.json:
- `npm run clean` - очистить кеш Vite и папку dist
- `npm run clean:cache` - очистить только кеш Vite
