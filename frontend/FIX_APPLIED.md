# ✅ ИСПРАВЛЕНИЯ ПРИМЕНЕНЫ

## Что было исправлено:

### 🔴 Главная проблема:
**JSON.stringify удалял поля со значением `undefined`**, поэтому cache key не менялся при изменении фильтров.

### ✅ Решение:
1. **Использование `null` вместо `undefined`** в cache key - теперь все поля попадают в JSON
2. **Принудительная инвалидация кеша** при изменении cacheKey
3. **Отключено кеширование** в React Query (`gcTime: 0`)
4. **Улучшена нормализация** фильтров для стабильного сравнения
